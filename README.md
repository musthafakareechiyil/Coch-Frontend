# Frontend

This is the React frontend for the Survey System built using Vite and Tailwind CSS. It connects with the Rails backend to allow users to view surveys, participate, and track progress.

## 🧰 Tech Stack

- **React** (Vite)
- **Tailwind CSS**
- **React Router DOM**
- **Axios**
- **Node**: 20.13.1
